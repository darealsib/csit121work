Site Evaluation

Website Name: The Humane Society of the United States

URL: https://www.humanesociety.org/

The Humane Society works to protect animals nationwide through rescue missions, advocacy, and education.

First Impression

The Humane Society website looks professional and caring. It uses emotional photos and organized navigation to connect users with its mission. Clear call-to-action buttons like “Donate” and “Adopt” make it easy to engage. However, the text on some banner images is hard to read, and better contrast could improve accessibility.

Results

After running the WAVE Accessibility Checker, the site showed 3 errors and 3 contrast errors, along with 15 alerts and 23 features. The errors mainly involve missing alt text and low contrast between text and images. Despite this, the website uses a strong ARIA structure (223 elements) and 57 structural elements that help assistive technologies interpret the layout effectively. The overall accessibility and usability are solid but could improve with a few design refinements.

Recommendations to Improve the Website

  • Increase contrast between text and background images.
  • Ensure all meaningful images include descriptive alt text.
  • Fix redundant or mislabeled ARIA attributes.
  • Add stronger focus indicators for keyboard navigation.
