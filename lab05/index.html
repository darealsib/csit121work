Reflection

Over the course of these lessons I came to appreciate why web standards and HTML matter. Standards help make sites easier to use, faster to load, and safer for visitors. HTML provides a simple, shared structure for content so different developers and browsers can present the same information consistently. Following these conventions reduces surprises for users and makes the web more reliable for everyone.
