The Web Share API

The Web Share API lets a website open the device’s built-in share menu. It can share text, links, and supported files using apps already on the device, like Messages, WhatsApp, or Email. This makes sharing fast because users don’t need to copy and paste anything. The feature works on most modern browsers, especially on mobile devices.

The goal is simple: give websites the same kind of sharing experience that mobile apps already have. With one tap, users can send content directly through apps they use every day.

Learn more from: W3C Web Share API Recommendation and MDN Web Share API.
